Module 4 · Lesson 4.2

Pattern Recognition

Abstract and inductive reasoning — identifying visual rules in shape sequences under time pressure. The questions that feel like guesswork until you learn to see systematically.

1

What These Tests Actually Look Like

Abstract reasoning — also called inductive reasoning — presents a sequence of shapes and asks you to identify the governing rule. You then select the next shape from four options, or identify which shape doesn't belong to a set.

These questions appear in SHL Inductive Reasoning (Barclays, Deloitte, Rothschild), Aon Scales (BNP Paribas, UBS, Schroders), Cappfinity (HSBC), and Korn Ferry Aspects (Citi, Macquarie). Typically 18–24 questions in 12–25 minutes — meaning you have 40–75 seconds per question.

Why These Feel Hard

The difficulty doesn't come from any single rule. Most rules are simple — a shape rotates, elements are added, a fill alternates. The difficulty comes from multiple rules operating at once, combined with answer options that each satisfy some of the rules but only one satisfies all of them. Every wrong option corresponds to a candidate who spotted one rule but missed another.

This lesson starts with a straightforward warm-up to establish the fundamentals, then rapidly escalates to the kind of multi-rule problems that actually appear in assessments. Five categories of rules account for virtually every question: rotation, reflection, element addition/removal, shading changes, and — most commonly at real test difficulty — combinations of two or three of these applied simultaneously.

Two-Rule Patterns: Where Tests Get Serious

At real assessment difficulty, most questions layer two rules on top of each other. A shape might rotate while its fill alternates, or elements are added while their position shifts. Candidates who spot only one rule pick a distractor that satisfies that rule but violates the other.

The Distractor Trap

A test would include: (A) Filled triangle pointing Up ✔, (B) Empty triangle pointing Up — correct rotation but wrong shading, (C) Filled triangle pointing Right — correct shading but wrong rotation, (D) Empty triangle pointing Down — both wrong. Candidates who spot only rotation pick (B). Candidates who spot only shading pick (C). Both rules must match.

4

Three-Rule Patterns: Expert-Level Questions

The hardest questions in any abstract reasoning test layer three independent rules onto a single sequence. Each answer option violates exactly one rule — meaning it looks almost right. The correct answer is the only option that satisfies every rule simultaneously. These questions separate the 80th+ percentile candidates from everyone else.

Worked Example 5 — Triple Combination Three rules govern this sequence. Identify all three and predict Frame 5.
→
→
→
→
1Rule A — Count: 1 → 2 → 3 → 4 → 5 squares
2Rule B — Shading: Filled → Empty → Filled → Empty → Filled (alternating)
3Rule C — Gold dot position: Top → Right → Bottom → Left → Top (rotates 90° CW)
4Answer: 5 filled squares with gold dot at top-centre

How a test would trap you:

Option (B): 5 empty squares, dot at top — wrong shading. Option (C): 5 filled squares, dot at right — wrong dot position. Option (D): 4 filled squares, dot at top — wrong count. Each option satisfies exactly two of the three rules.

Worked Example 6 — 3×3 Matrix In some tests, patterns appear in a 3×3 grid. The rule applies across rows AND down columns. What replaces the missing cell?

Each row and column follows the same logic: the third cell combines or resolves the first two.

1Row rule: Column 1 = filled shape. Column 2 = outline of a different shape. Column 3 = Column 1 shape with Column 2 shape cut out (white) inside it.
2Row 1: Filled circle + outline square → circle with white square inside ✓
3Row 2: Filled square + outline triangle → square with white triangle inside ✓
4Row 3: Filled triangle + outline circle → Filled triangle with white circle cut out inside it
Answer

Column verification: Column 1 (circle, square, triangle) = each shape appears once as filled. Column 2 (square, triangle, circle) = each shape appears once as outline. Column 3 = each combination appears once. The rule is consistent in both directions.

Matrix Technique

For 3×3 grid questions, check the rule across rows first, then verify it holds down columns. If your answer satisfies the row rule but breaks the column rule, it's wrong. The correct answer always satisfies both. This double-verification is time-consuming but prevents the most common errors on the hardest questions.

5

The Systematic Approach

Don't stare at the sequence hoping inspiration strikes. Run through this checklist for every element in the frame. Most candidates fail because they check the main shape but miss a secondary marker (a dot, a line, a shading change) that carries a separate rule.

The 5-Point Pattern Checklist
1. SHAPE — Does the main shape change type?
2. POSITION — Does anything rotate, shift, or reflect?
3. SIZE — Does any element grow or shrink?
4. SHADING — Does fill change (filled / striped / empty)?
5. COUNT — Are elements added, removed, or rearranged?
Run this for EACH distinct element — main shape, inner shape, dots, lines, markers. On hard questions, different elements follow different rules.
Why Intuition Fails

Test designers create distractors that feel right. Each wrong option satisfies one or two rules — enough to trigger a sense of recognition. Candidates who rely on gut feeling achieve 40–50% accuracy on hard questions (barely above the 25% chance of random guessing on 4 options).

Why the Checklist Works

The checklist forces you to identify every rule before evaluating options. Explicitly state the rules: "main shape rotates 90° CW, fill alternates, dot moves anticlockwise." Then test each option against all rules. Only one option satisfies them all.

Time Allocation — 60 Seconds

10 seconds: scan the sequence, note how many distinct elements exist. 20 seconds: run the checklist, state the rule(s) explicitly. 20 seconds: test each option against your rules. 10 seconds: confirm by checking the answer against every frame. If you can't identify the rule within 30 seconds, eliminate one or two options, pick the most likely, and move on.

Practice Question 3
Which figure does NOT belong in this group?
  • (A)
  • (B)
  • (C)
  • (D)
Show Answer ▾
1Count sides and dots: (A) 4 sides, 2 dots. (B) 6 sides, 3 dots. (C) 8 sides, 4 dots. (D) 5 sides, 2 dots.
2Rule: Number of dots = number of sides ÷ 2. A: 4/2=2 ✓. B: 6/2=3 ✓. C: 8/2=4 ✓. D: 5/2=2.5, but has 2 dots.
3Answer: (D) — the pentagon has an odd number of sides, breaking the sides÷2=dots relationship

Why this is hard: The initial instinct is to count dots alone (2, 3, 4, 2 — D looks like it could belong as "another shape with 2 dots"). The real rule links dots to sides, and only even-sided polygons satisfy it. This requires noticing a relationship between two properties, not just tracking one property.

Practice Question 4
What completes the bottom-right cell of this 3×3 matrix?
  • (A)
    Triangle with 3 dots (2 top, 1 bottom)
  • (B)
    Triangle with 3 dots (1 top, 2 bottom)
  • (C)
    Square with 3 dots
  • (D)
    Triangle with 4 dots
Show Answer ▾
1Row rule (across): Each row keeps the same outer shape. Dot count increases left to right: 1 → 2 → 3.
2Column rule (down): Each column keeps the same dot count. Outer shape changes: Circle → Square → Triangle.
3Dot arrangement: In every cell across the grid, 3-dot configurations use the same layout: 2 at top, 1 at bottom.
4Missing cell = Row 3 (triangle) + Column 3 (3 dots, arranged 2-top 1-bottom) = (A)

Trap analysis: (B) has 3 dots in a triangle but inverted arrangement (1-top, 2-bottom) — breaks the dot-layout consistency. (C) is a square, not a triangle — wrong outer shape for Row 3. (D) has 4 dots — wrong count for Column 3.

Key Takeaways

  • At real assessment difficulty, most questions layer two or three rules simultaneously. Single-rule questions are rare beyond warm-up level.
  • Every wrong answer option satisfies some rules but violates at least one. The correct answer is the only option that satisfies all rules at once.
  • Run the 5-point checklist (Shape, Position, Size, Shading, Count) for every distinct element in the frame — main shapes, secondary markers, dots, lines.
  • For 3×3 matrix questions, verify your answer satisfies the rule across its row AND down its column. If it only works in one direction, it's wrong.
  • Rotation vs Reflection: rotation preserves handedness, reflection reverses it. Track one asymmetric feature across frames to tell them apart.
  • Time rule: if you can't identify the rule within 30 seconds, eliminate what you can, pick the best remaining option, and move on. One slow question ruins three fast ones.
